Aprendendo HTML: Um Guia Completo

HTML (HyperText Markup Language) é fundamental para a construção de páginas na web. Ao longo deste documento,
você aprenderá os principais elementos utilizados na criação de um site moderno. HTML permite que você defina a estrutura de qualquer
página da web,usando vários elementos.

Estrutura Básica

A primeira coisa que se deve aprender ao criar páginas na web é como utilizar corretamente as tags de cabeçalhos.
A tag < h1> define o título principal da página, enquanto < h2> a < h6> são usadas para subtítulos com diferentes níveis de hierarquia.
  1. Cabeçalhos são úteis para organizar o conteúdo de forma hierárquica.
  2. Parágrafos, por outro lado, organizam o texto em blocos.
  3. Quebras de linha < br> podem ser usadas quando se quer uma nova linha sem começar um novo parágrafo.
